# NOTES:
# 1. All distance units unless otherwise noted are in meters.
# 2. A Mass of >= 1,000,000.0 is considered infinite
# 3. Collision priorities are in order where the highest value wins.
# 4. The higher bullet restitution is the more a player's shots will knock an
#    enemy around.
# 5. PlayerRestitution is currently unused, but was supposed to be the amount of kick
#    applied to the enemy after colliding with the player.
# 6. Rotation speeds(ScanSpeed, TurnRate) are in revolutions / sec., not radians / sec.
# 7. ScanSpeed is currently unused.
# 8. FOVWidth is in degrees, not radians.  
# 9. MaxElevDiffSeen is a difference of altitude (in m), not angle
# 10. TurnError is in degrees, not radians (currently unused, though)
# 11. Speeds are in m / s.
# 12. Accelerations (and decelerations) are in m / s / s.
# 13. MainProjReload is the reload time for a single shot within a burst. (in sec.)
# 14. MainProjBurstLength is the number of shots / burst.
# 15. MainProjBurstReload is the reload time between bursts. (in sec.)
